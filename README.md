# Lógica de Programação

### Visão Geral

<p>Este projeto foi desenvolvido como parte do curso de lógica de programação, com foco no aprendizado de conceitos fundamentais de desenvolvimento web e programação em JavaScript. A estrutura do projeto está organizada para fornecer uma progressão gradual no aprendizado, permitindo que o aluno pratique conceitos como variáveis, condicionais, loops, interações dinâmicas e manipulação de DOM.</p>

### Descrição das Pastas e Arquivos

1. **Pasta `Day-1` a `Day-4`**

    <p>Cada uma dessas pastas contém exercícios e aplicações práticas que evoluem gradualmente em complexidade:</p>

- **Arquivos Comuns:**
    - `index.html`: Estrutura base do HTML, com elementos de interface simples e centralizados.
    - `style.css`: Estilos visuais, incluindo gradientes, fontes externas e layout responsivo.
    - `app.js`: Lógica em JavaScript para interatividade e funcionalidade.
- **Conteúdo por Dia:**
    - `Day-1`:
        - Introdução ao JavaScript com `alert`, `prompt`e interações básicas.
        - Lógica simples como exibir mensagens de boas-vindas.
    - `Day-2`:
        - Incremento na lógica com loops (`for`, `while`) e condicionais básicas.
        - Introdução a validações de entrada.
    - `Day-3`:
        - Trabalha com operações matemáticas, loops mais avançados e interações dinâmicas.
    - `Day-4`:
        - Foco em números aleatórios e aplicações práticas de lógica condicional.

</br>

---

</br>

2. **Pasta `Desafio`**

    <p>Esta pasta contém exercícios e desafios organizados para consolidar os aprendizados.</p>

- **Subpasta `Desafios`:**

    - Conjunto de exercícios simples que abordam fundamentos como:
        - Operações matemáticas (soma, subtração).
        - Condicionais (`if-else`) e loops.
        - Interações com o usuário via `prompt`, `alert` e `console.log`.

    - Exemplos:
        - `1-app.js`: Exibe uma mensagem de boas-vindas no console.
        - `5-app.js`: Realiza a soma de dois números.
        - `8-app.js`: Determina se um número é positivo, negativo ou neutro.

- **Subpasta `Desafio-final`:**

    - Desafios mais avançados que integram conceitos:
        - Geração de números aleatórios.
        - Aplicações práticas com lógica mais complexa.
        - Uso extensivo de loops e condicionais.

- **Arquivos Comuns na Pasta `Desafio`:**

    - `index.html` e `style.css`: Fornecem uma base para expandir os desafios com interface visual.
    - `app.js`: Ponto central para executar a lógica dos desafios.

</br>

### Funcionalidades Desenvolvidas

- **Interação Dinâmica:**

    - Uso de `alert`e `prompt`para obter entradas e exibir mensagens.
    - Geração de números aleatórios com `Math.random`.
    - Validações simples para garantir respostas corretas.

- **Interface Responsiva:**

    - Gradientes, sombras e fontes modernas fornecem uma experiência visual agradável.
    - Layout centralizado para exibição clara das mensagens e elementos visuais.

- **Lógica e Matemática:**

    - Aplicação de condicionais e loops para resolver problemas práticos.
    - Operações matemáticas (soma, subtração) e controle de fluxo.

### Sugestões para Uso

1. **Como Navegar:**
    - Comece pelo `Day-1` para entender os fundamentos e avance para os dias seguintes.
    - Experimente os desafios na pasta `Desafio` após completar os exercícios básicos.
2. **Executar os Scripts:**
    - Abra os arquivos `index.html` no navegador para ver os resultados visuais.
    - Utilize o console do navegador para acompanhar as saídas de `console.log`
3. **Customizar:**
    - Modifique os scripts para explorar diferentes cenários e desenvolver lógica personalizada.
    - Tente integrar os desafios em uma única interface visual para praticar manipulação de DOM.
